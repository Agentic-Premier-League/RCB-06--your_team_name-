About the Role

As a Software Engineering Intern at Google, you'll be embedded directly in a product team — contributing to real features that millions of people use every day. You'll work side-by-side with senior engineers, product managers, and designers across the full development cycle: from design and coding to testing and launch.

This role is within our Core Infrastructure team, working on distributed systems that power Google Search and Google Cloud. You'll get hands-on experience with large-scale data pipelines, backend services, and reliability engineering.

At the end of the internship, you'll present your project to senior leadership and receive a comprehensive performance review. Strong interns are regularly extended full-time return offers.

What You'll Do

Design and implement features for large-scale distributed systems
Write clean, testable code in Python and Go with code reviews
Collaborate with engineers across teams in Agile sprints
Participate in on-call rotations and incident response exercises
Present your project and learnings at the intern showcase

Requirements

Must Have

Currently enrolled in a BS/MS/PhD program in CS or related field
Strong proficiency in Python and/or one other systems language (Go, C++, Java)
Understanding of data structures, algorithms, and distributed systems fundamentals
Experience with REST APIs and SQL databases
Familiarity with Agile development practices and version control (Git)

Nice to Have

Experience with Kubernetes or Docker container orchestration
Contributions to open-source projects
Familiarity with TypeScript or React for full-stack work
Previous internship or research experience at scale
